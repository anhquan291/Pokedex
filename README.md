## Get Started

```bash
# install expo cli
npm install -g expo-cli
```
``` bash
# install dependencies
npm install
```
Or use `yarn`
``` bash
yarn install
```
``` bash
# run project
expo start
```


#### Feature
  - Pokemon Detail
  - Abilities 
  - Movements
  - Search
